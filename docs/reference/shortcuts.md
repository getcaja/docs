# Keyboard Shortcuts

## General

| Shortcut | Action |
|----------|--------|
| `⌘Z` | Undo |
| `⌘⇧Z` | Redo |
| `⌘S` | Save |
| `⌘⇧S` | Save As |
| `⌘O` | Open file |
| `⌘\` | Toggle Layers Panel |
| `⌘⇧\` | Toggle Properties Panel |
| `⌘⇧A` | Toggle advanced mode |
| `⌘⇧P` | Toggle preview mode |

## Canvas

| Shortcut | Action |
|----------|--------|
| `V` | Pointer tool |
| `F` | Frame tool |
| `T` | Text tool |
| `Escape` | Return to Pointer |
| `⌘+` | Zoom in |
| `⌘-` | Zoom out |
| `⌘0` | Reset zoom to 100% |

## Selection & Editing

| Shortcut | Action |
|----------|--------|
| `⌘A` | Select all siblings |
| `⌘C` | Copy |
| `⌘X` | Cut |
| `⌘V` | Paste |
| `⌘D` | Duplicate |
| `⌘G` | Group |
| `⌘⇧G` | Ungroup |
| `F2` | Rename selected |
| `Delete` | Delete selected |
| `↑` `↓` | Reorder within parent |
| `←` `→` | Collapse/expand (column layouts) |
| `Escape` | Deselect |

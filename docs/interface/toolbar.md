# Toolbar

The bottom toolbar provides quick access to creation tools, viewport controls, and preview mode.

## Sections

### Add Elements
The `+` button opens a menu to insert new elements: Frame, Text, Link, Image, Button, Input, Textarea, Select.

### Canvas Tools
Three tool buttons — **Pointer (V)**, **Frame (F)**, **Text (T)**. The active tool is highlighted.

### Viewport
A dropdown to switch responsive breakpoints: Full, Tablet (768px), Mobile (375px).

### Preview
The eye icon (`⌘⇧P`) toggles preview mode, which disables editing and enables link navigation.

::: tip
In preview mode, the toolbar collapses — only the viewport selector and preview toggle remain visible.
:::
